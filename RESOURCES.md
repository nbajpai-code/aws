# AWS Learning Resources

## 📚 Official Materials
*   **[AWS Skill Builder](https://skillbuilder.aws/)**: The official learning center. Contains 600+ free digital courses, labs, and the "Cloud Quest" game.
*   **[AWS Whitepapers & Guides](https://aws.amazon.com/whitepapers/)**: Essential reading for the Solutions Architect exams. Start with the "Well-Architected Framework".
*   **[AWS Documentation](https://docs.aws.amazon.com/)**: The source of truth for every service.

## 🛠️ Labs & Practice
*   **[AWS Free Tier](https://aws.amazon.com/free/)**: 12 months free for new accounts. Essential for hands-on practice.
*   **[Tutorials Dojo (Jon Bonso)](https://tutorialsdojo.com/)**: widely considered the **best practice exams** in the industry.
*   **[Adrian Cantrill's Courses](https://learn.cantrill.io/)**: In-depth, visual courses that focus on real-world skills, not just passing the exam.

## 🎮 Gamified Learning
*   **[AWS Cloud Quest](https://aws.amazon.com/training/digital/aws-cloud-quest/)**: Learn AWS by playing a role-playing game.
*   **[AWS Card Clash](https://aws.amazon.com/training/digital/card-clash/)**: Learn architecture via a card game.
