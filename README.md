## 🚀 Getting Started
1.  Create a free **[AWS Skill Builder](https://skillbuilder.aws/)** account.
2.  Play **Cloud Quest** to learn the basics in a fun way.
3.  If aiming for **Solutions Architect Associate**, check out [Adrian Cantrill's course](https://learn.cantrill.io/).
